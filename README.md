## Getting Started

To get a local copy up and running follow these simple example steps.

### Installation and run Client

1. Clone the repo

```sh
git clone https://github.com/vlasiuk-anatolii/delivery-app
```

2. Install NPM packages

```sh
npm install
```

3. Check that const BASE_URL is exactly http://127.0.0.1:5000 in src/api/api.ts

```JS
const BASE_URL = 'http://127.0.0.1:5000';
```
4. Type in the terminal

```sh
npm start
```
5. After that client will start on http://127.0.0.1:3000

### Installation and run Server

1. Clone the repo

```sh
git clone https://github.com/vlasiuk-anatolii/delivery-app
```

2. Install NPM packages

```sh
npm install
```
3. Type in the terminal

```sh
npm start
```
4. After that client will start on http://127.0.0.1:5000

### Installation DataBase

1. File "db.sql" for restore database exists in the folder db which you get when cloned repo

2. Install on your computer PostgreSQL

3. Creat empty database with name Products

4. You should used utility of PostgreSQL such as pg_restore and restore data base with specified name.

5. Restored database must consists 2 tables: 'shops', 'products'.
